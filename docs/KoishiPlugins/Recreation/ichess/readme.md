# 国象

[![npm](https://img.shields.io/npm/v/koishi-plugin-ichess?style=flat-square)](https://www.npmjs.com/package/koishi-plugin-ichess)

基于 [chess.js](https://github.com/jhlywa/chess.js) 的国际象棋插件

## 插件功能

- [x] 【创建/加入】对局
- [x] 悔棋
- [x] pgn 导出
- [x] 查看棋盘
- [x] 停止下棋

- [ ] 玩家头像显示（正在制作）
- [ ] 用时计算（正在制作）

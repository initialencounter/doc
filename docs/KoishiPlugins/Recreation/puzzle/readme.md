# 华容

[![npm](https://img.shields.io/npm/v/koishi-plugin-puzzle?style=flat-square)](https://www.npmjs.com/package/koishi-plugin-puzzle)

拼图游戏，数字华容道

原游戏: [扫雷联萌](http://tapsss.com)

## 指令

| 描述          | 示例                     |
| ------------- | ------------------------ |
| 开始 puzzle   | pz 4                     |
| 加入 puzzle   | pz.join                  |
| 退出 puzzle   | pz.quit                  |
| 自定应 puzzle | pz.def + puzzle 导出数据 |
| puzzle 排行榜 | pz.rank                  |

gocqhttp-dev

版本 1.1.1-dev-0.4.2 和 1.1.1-dev-0.4.5 存在致命bug, 请尽快更新

以下是最新版本的readme：

适配 gocqhttp-dev 启动器

本插件魔改自koishi-plugin-gocqhttp

本插件与gocqhttp不可同时运行, 启用前请先停止gocqhttp

部署 SignServer 点我

协议版本抄自MrXiaoM/qsign

目前支持以下功能：

  • 扫描二维码登录
  • 手机短信验证码
  • captcha 图片
  • 滑条验证
  • 手动输入ticket
  • 配置签名服务器
  • 检测qsignServer, 减少冻结风险
  • 监听多个服务，可用于对接其他框架，如 Nonebot,Zerobot-plugin等
  • 设置登录设备

使用方法​

要使用此插件，需要完成这些配置：

  1. 启动 qsign, 有两种方法
  • 方法一，内置qsign: 安装@yunkuangao/qsign，并启用
  • 方法二，外置qsign：部署 SignServer 点我
  1. 启用本插件
  2. 前往adapter-onebot, 填写qq账号和密码，勾选启用自动创建 gocqhttp 子进程，启用 adapter-onebot
